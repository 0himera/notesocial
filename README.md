# note*me

Ультра-лёгкая статическая соцсеть-заметки в стиле iCloud Notes.

## Архитектура

- **Чтение** — публичный статический HTML
- **Постинг** — через скрытую админку с SHA-256 авторизацией

## Файлы

```
noteme/
├── data.json        # Хранилище данных
├── build.js         # Генератор статических страниц
├── api/post.js      # Vercel Serverless API
├── public/admin.html # Админка для публикации
├── dist/            # Сгенерированные HTML (после build)
└── vercel.json      # Конфигурация деплоя
```


## Деплой на Vercel

1. Создайте репозиторий и загрузите файлы
2. Подключите к Vercel
3. Добавьте переменную `VERCEL_DEPLOY_HOOK` (опционально)
4. Deploy!

## Как это работает

1. `build.js` читает `data.json` и генерирует статические HTML:
   - `index.html` — список всех юзеров (как заметки в iCloud)
   - `{userId}.html` — страница заметок юзера

2. Админка (`admin.html`) позволяет:
   - Создать нового пользователя (+ пароль)
   - Добавить заметку (требуется пароль)

3. API (`api/post.js`) принимает запросы:
   - Сверяет SHA-256 хеш пароля с хранимым
   - Обновляет `data.json`
   - Дёргает Deploy Hook для ребилда

## Безопасность

- Пароли хранятся как SHA-256 хеши
- Хеширование происходит на клиенте
- Сервер только сравнивает хеши

## ~2KB Rule ✓

- Никаких внешних библиотек
- Системные шрифты
- Inline CSS
- Минималистичный дизайн
